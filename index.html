SMS Caster for bedrifter

SMSCASTER.NO gateway gir forskjellige tekniske løsninger for norske organisasjoner og bedrifter.

 

Funksjoner

Kraftige løsninger for bedriftens behov

Ved bruk av SMSCASTER.NO SMS Gateway og SMS Web Portal har bedrifter mulighet til å klargjøre SMS på forhånd og sette utsendelse frem i tid.

  • Kodeord

    Motta SMS på nett ved hjelp av kodeord. Kodeord er et ord som man bruker til å motta innkommende SMS fra kunder på et kortnummer og kan brukes f.eks. i rabattkuponger, SMS Konkurranser, SMS-Varsling, dialog med kunder i Kundeklubber. Man kan lage egen Kodeord i SMSCASTER.NO SMS Gateway for å ha en effektiv og raskt toveis kommunikasjon med kunder.

  • SMS Gateway API

    SMS Gateway API er en tjeneste for utviklere og brukes for å integrere IT-systemer til SMSCASTER.NO server. SMSCASTER.NO API er en REST-API grensesnitt. Bedrifter får API bruker for API call etter at kundekonto opprettes hos SMSCASTER.NO SMS gateway.

Våre kunder

SMSCASTER.NO SMS tjenester

SMSCASTER.NO SMS tjenester er en nettbasert skyløsning som ved hjelp av nyeste teknologi tilbyr et pålitelig system som kan tilpasses til ulike typer behov hos små og store bedrifter.

SMS til grupper

SMSCASTER.NO SMS løsning gir mulighet til umiddelbart utsendelse av melding til alle medlemmer av kundegruppen eller flere kundegrupper. Med SMSCASTER.NO SMS løsning er det lett å organisere kontakter i flere grupper.

Velg egen avsender

Med SMSCASTER.NO SMS løsning er det mulig til å velge egen avsender for din bedrift. Egendefinert avsender vises som sender på mobile enheter til kundene. Avsender kan bestå av tekst eller tall, evt. en kombinasjon av tekst og tall av maks 11 karakter. Navn til avsender skal godkjennes fra SMSCASTER.NO først.

Import av kontakter fra fil

I SMSCASTER.NO SMS Gateway er det mulig å importere alle kundekontakter umiddelbart fra en CSV fil og spare tid i å legge de enkeltvis.
